Compile & Run:
==============
The program is designed to compile under command line tool of stdlinux environment.

In terminal, direct to the folder which contains Tokenizer.java.

Compile Tokenizer.java by using "javac Tokenizer.java"

Run the executable file by inputting "java Tokenizer INPUT_FILE", where INPUT_FILE must exist.
